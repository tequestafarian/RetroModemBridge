# Local BBS Search and Paging

This build adds search and paging to the local BBS menu on the retro computer side.

## From the retro terminal

```text
ATDT MENU
1
```

This opens the BBS Directory.

Inside the directory or favorites list:

```text
N   Next page
P   Previous page
S   Search
C   Clear search
M   Main menu
Q   Disconnect
01-15 Dial the listed entry on the current page
```

## Search

Press `S`, then type a search term such as:

```text
coco
ansi
mystic
games
online
```

The search checks name, alias, host, category, system type, notes, and last result.

## Notes

The Windows app does all the filtering and paging. The retro computer only needs the same serial terminal program as before.
