# Commercial-Style Release Summary

## RetroModem Bridge v3 Beta 3

RetroModem Bridge v3 Beta 3 is the largest update to the project so far.

This release turns RetroModem Bridge into a full retro-computing BBS connection center with a local ANSI BBS menu, searchable/paged BBS directory, retro-side Import Guide browser, Monthly/Daily guide updater, Session Mirror, connection testing, profiles, support bundle export, and a new app icon.

## One-line summary

RetroModem Bridge v3 Beta 3 lets vintage computers browse, update, manage, and dial Telnet BBSes through a Windows-powered local BBS-style modem bridge.
