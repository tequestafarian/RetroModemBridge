# Daily Guide Download Notes

If the Daily Guide download fails, likely causes are:

- the download page returned a verification / bot-check page
- the page layout changed and the app could not identify the Daily ZIP link
- the server blocked the request
- the daily ZIP was temporarily unavailable

The Windows updater still includes **Load ZIP/CSV manually** as a fallback.

The Import Guide window now has a **New only** filter so after any successful Monthly update, Daily update, or manual ZIP/CSV load, you can view only guide entries that are not already in your BBS Directory.
